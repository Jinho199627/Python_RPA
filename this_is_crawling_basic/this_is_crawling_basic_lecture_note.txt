'이것이 크롤링이다' 기본 편 강의

라이브러리: 도구들

<크롤링을 위한 정말 쉬운 HTML>
    HTML : 웹 페이지의 구조를 나타내기 위한 영어
    태그(웹페이지의 구성 요소)로 구성되어 있다. < > 로 구분
    시작태그와 종료태그가 있다.

    <태그명> 내용 </태그명>
    <h1> 스타트코딩 </h1>

    태그에는 속성명(주황)과 속성값(파랑)이 있을 수 있다.
    <태그명 속성명 = "속성값"> 내용 </태그명>
    <h1 id = "title"> 스타트코딩 </h1>
    h1이라는 태그에 title이라는 별명을 붙인다.

    태그명: div // 역할: 구역 나누기(댓글을 위한 구역, 광고를 위한 구역 등등)
    태그명: a // 역할: 링크(제목을 클릭하면 기사가 있는 페이지로 이동한다.)
    태그명: h1 // 역할: 제목(중요한 내용들)
    태그명: p // 역할: 문단
    태그명: ul, li // 역할: 목록(리스트 형식 데이터)

    매우 중요한 내용
    <div class = "news_info"> # div 태그는 a 태그들의 부모 태그이다.
        <a href="주소1.com></a> # a 태그는 div 태그의 자식 태그이다.
        <a href="주소2.com></a> # a 태그는 클릭하면 이동하게 해주는 역할을 한다.
        <a href="주소3.com></a>
    </div>

    크롬으로 네이버 뉴스를 열고 F12번을 눌러 HTML 코드를 연다음 요소와 해당 코드를 찾아주는 아이콘을 통해 특정 코드 구문으로 이동한다.
    그 후 해당 부분의 문자열을 원하는 내용으로 바꿔줄 수 있다.

<request로 HTML을 파이썬으로 가져오기> - warm_up.py 참고
    request: HTTP 통신을 위한 파이썬 라이브러리(프로그램 개발을 쉽게 하기 위한 도구)
    HTTP 통신: 인터넷에 접속할 때 발생하는 요청과 응답들
        get: 뉴스 1페이지 보여주세요.
        post: 특정 정보들과 함께 말을 건다. 예를 들어 아이디 비밀번호를 주고 응답을 요청
    외부 라이브러리(기본적으로 설치된 게 아닌 라이브러리): pip
        pip install request를 cmd 터미널에 입력

<beautifulsoup를 통해 원하는 태그를 선택하자> - warm_up.py 참고
    beautifulsoup: HTML 분석을 위한 파이썬 라이브러리
    pip install beautifulsoup4 로 설치
    사용방법: beautifulsoup(html 코드, html 번역 선생님)

<크롤링에서 가장 중요한 CSS 선택자>
    CSS: 웹사이트의 디자인을 표시하기 위한 언어, 글자 색 변경, 폰트 크기, 가로 세로 길이 변경
    CSS 선택자: 디자인을 변경할 HTML 태그를 선택하는 것 = 크롤링할 HTML 태그를 선택하기
    선택자를 자유자재로 활용할 수 있어야 크롤링이 쉽다.
    태그 선택자: 태그를 이름으로 선택하다.
        <h1>제목입니다.</h1>    선택자: h1
    id 선택자: id 속성 값이 있을 때 id 값으로 선택한다.
        <div id = "articleBody">본문 내용입니다.</div>  선택자: #articleBody
    class 선택자: class 값으로 선택한다. 태그 그룹의 별명
        <div class = "info_group">뉴스 목록</div>   선택자: .info_group
    자식 선택자: 원하는 태그에 별명이 없을 때 사용한다. 별명이 없을 땐 부모 클래스를 활용한다.
        <div class = "logo_sports"><span>스포츠</span></div>    선택자: .logo_sports>span
